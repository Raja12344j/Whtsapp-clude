WhatsApp Cloud Panel (Demo)
---------------------------
1) Copy .env.example to .env and set WHATSAPP_PHONE_ID and WHATSAPP_ACCESS_TOKEN
2) npm install
3) npm start
4) Open http://localhost:3000 in your browser
Notes:
- This is a demo panel for the official WhatsApp Cloud API.
- Do NOT run this on a mobile device. Use a desktop or VPS.
- For production, add proper error handling, persistent storage, authentication and rate limiting.
